Server / PHP integration (optional)

यदि आप इसे PHP template में लगाना चाहें — तो आप portrait image को `/assets/images/portrait.png` में रखकर नीचे दिए example का उपयोग कर सकते हैं:

<?php
$img = '/assets/images/portrait.png';
?>
<img src="<?= $img ?>" alt="Portrait" />

और server-side resizing के लिए आप PHP में GD या Imagick इस्तेमाल कर सकते हैं — बताइए मैं ready-made code भी दे दूँ।
